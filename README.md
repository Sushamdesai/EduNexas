EduNexas is a comprehensive e-learning platform developed using the MERN Stack (MongoDB, Express.js, React.js, Node.js). The platform allows users to register, log in using OTP authentication, purchase courses, subscribe to content, and start learning. This project aims to provide an interactive and seamless learning experience.

Features
User Registration and Login: Secure OTP authentication for user registration and login.
Course Purchase and Subscription: Integration with Razorpay for purchasing courses and subscriptions.
Interactive Learning: Access to a variety of interactive courses.
User Dashboard: Personalized user dashboard to manage courses and subscriptions.
Admin Dashboard: Administrative features for managing users, courses, and transactions.
Tech Stack
Frontend: React.js
Backend: Node.js, Express.js
Database: MongoDB
Authentication: OTP-based authentication
Payment Gateway: Razorpay
